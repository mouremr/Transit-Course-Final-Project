# Transit-Course-Final-Project


Upload of the final project I did for the Modeling Smart Transportation Systems course I took at the Technical University of Berlin in July of 2024. The goal of the project was to create a stock flow diagram using vensim which showed how various factors might be related to the overall use of different transportation methods, and explore how changing the different variables we included in our simulation affected the number of people willing to take different forms of transit.
